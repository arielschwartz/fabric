#Fabric

Fabric is a minimal and responsive theme for Octopress.

##Install

Type the code below in terminal.

	$ cd octopress
	$ git clone git://github.com/panks/fabric.git .themes/fabric
	$ rake install['fabric']
	$ rake generate

Have problems when installing with zsh? Try `rake install\['fabric'\]` instead.

##Features

- Clean design
- Tapirgo search
- Responsive layout
- Ajaxified pages
- and many more..

Read [here] (http://panks.me) for detail on installation and setup.
